# HDU我爱记单词脚本

## 脚本说明
本脚本仅供个人学习编程使用，绝无引导作弊行为！使用作弊者后果自负！

## 简介

## 环境要求
- Python 3.x
- 相关Python库
   - request

### 安装依赖
在命令行中运行以下命令来安装所需的Python库：
```bash
pip install request
```

## 使用方法

**修改token**: 修改文件开头的token, token通过打开https://skl.hdu.edu.cn/, 通过浏览器开发工具(按下F12), application/local storage/https://skl.hdu.edu.cn/sessionId获取

**运行脚本**：
   在命令行中进入脚本所在目录，运行以下命令：
   ```bash
   python word.py
   ```
然后根据指引使用即可

空输入使用默认值

脚本有三个功能: **创建自测**,  **构建题库**, **答题** 
- 注: 项目已经提供的题库为作者不保证正确率， 因为每周的题库并不完全相同

### 答题

应先在自测中测试分数, 检查题库命中率, 未命中次数将在运行结束后由no_hit给出, 若命中率过低或命中但答案错误,  应考虑更新或重建题库
- **不要不测试就直接考试, 获得0分才追悔莫及**

脚本对不在题库中的题会随机选择, 当题库命中次数大于等于分数下界后, 脚本会随机选择完成后续题目无论是否命中题库

### 创建自测

用于获取构建题库用的自测

由于申请测试的时间最短为五分钟, n次自测的创建将花费约(n-1)*5min

### 构建题库

脚本将遍历自测的历史记录, 将新遇到的题目-答案组合加入旧题库, 若无data.json, 脚本将创建data.json用于保存题库
