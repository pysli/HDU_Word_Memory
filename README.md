# HDU我爱记单词脚本

## 脚本说明
本脚本仅供个人学习编程使用，绝无引导作弊行为！使用作弊者后果自负！

## 简介

## 环境要求
- Python 3.x
- 相关Python库：
  - selenium

### 安装依赖
在命令行中运行以下命令来安装所需的Python库：
```bash
pip install selenium
```

### 浏览器驱动
本脚本使用Edge浏览器，需要下载对应的`msedgedriver.exe`，并将其放在脚本同一目录下。下载地址：[Microsoft Edge WebDriver](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)

## 使用方法

**运行脚本**：
   在命令行中进入脚本所在目录，运行以下命令：
   ```bash
   python Script.py
   ```
脚本分为三个部分: **自动创建自测**,  **构建题库**, **答题**
# 答题脚本

启动脚本后在打开的浏览器内进入答题界面, 应可见题目和选项, **关闭自动下一题**, 后在命令行中向脚本输入提供非零输入即可自动运行
应先在自测中测试分数, 检查题库命中率, 未命中次数将在运行结束后由no_hit给出, 若命中率过低或命中但答案错误,  应考虑更新或重建题库

# 创建自测

启动脚本后进入有"开始自测"字样的界面, 在命令行向脚本键入创建自测的数量, 脚本将自动进行开始自测->提交退出的循环
由于申请测试的时间最短为五分钟, n次自测的创建将花费约(n-1)*5min

# 构建题库

启动脚本后进入有"开始自测"字样的界面, 提供任意非零输入, 脚本将遍历自测的历史记录, 将新遇到的题目-答案组合加入旧题库
若无data.json, 脚本将创建data.json用于保存题库


## 注意事项
- **关闭自动下一题**
- **页面结构变化**：如果HDU我爱记单词平台的页面结构发生变化，脚本中的XPath可能需要相应调整。
