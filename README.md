# HDU我爱记单词脚本

## 脚本说明
本脚本仅供个人学习编程使用，绝无引导作弊行为！使用作弊者后果自负！

## 简介

## 环境要求
- Python 3.x
- 相关Python库：
  - selenium

### 安装依赖
在命令行中运行以下命令来安装所需的Python库：
```bash
pip install selenium
```

### 浏览器驱动
本脚本使用Edge浏览器，需要下载对应的`msedgedriver.exe`，并将其放在脚本同一目录下。下载地址：[Microsoft Edge WebDriver](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)

## 使用方法

**运行脚本**：
   在命令行中进入脚本所在目录，运行以下命令：
   ```bash
   python Script.py
   ```
脚本分为三个部分: **自动创建自测**,  **构建题库**, **答题**, 若不构建题库,  仅使用答题脚本(word.py)即可
- 注: 项目已经提供的题库为371次自测后生成, 覆盖率约98%
### 答题脚本
- **脚本使用data.json作为题库, 若不自行构建题库, 请下载项目提供的题库使用**

启动脚本后在打开的浏览器内进入答题界面, 应可见题目和选项, 后在命令行中向脚本输入提供非空输入即可自动运行

若hit数较高, 第89-99题进行随机选择控制分数, 完成100道题后, 脚本会等待至剩余一分钟进行提交

应先在自测中测试分数, 检查题库命中率, 未命中次数将在运行结束后由no_hit给出, 若命中率过低或命中但答案错误,  应考虑更新或重建题库
- **不要不测试就直接考试, 获得0分才追悔莫及**

### 创建自测

用于获取构建题库用的自测, 启动脚本后进入有"开始自测"字样的界面, 在命令行向脚本键入创建自测的数量, 脚本将自动进行开始自测->提交退出的循环

由于申请测试的时间最短为五分钟, n次自测的创建将花费约(n-1)*5min

### 构建题库

**修改token**: 修改文件开头的token, token通过打开https://skl.hdu.edu.cn/, 通过浏览器开发工具(按下F12), 选择网络选项, 从中找到request headers中有X-Auth-Token的请求, 复制X-Auth-Token作为token

脚本将遍历自测的历史记录, 将新遇到的题目-答案组合加入旧题库, 若无data.json, 脚本将创建data.json用于保存题库

- 注: build_question_bank(old).py使用selenium模拟浏览器操作获取题库, 无需获取token, 但速度慢且不稳定容易崩溃, 不建议使用


## 注意事项
- **页面结构变化**：如果HDU我爱记单词平台的页面结构发生变化，脚本中的XPath可能需要相应调整。
